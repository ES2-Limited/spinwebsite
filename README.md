# SPIN Website

This is the official website for the **Sustainable Power Initiative Nigeria (SPIN)** project. It serves as a comprehensive portal for information about the project, its mandate, participating states, news, and resources.

## Features

- **Responsive Design**: Built with Tailwind CSS for optimal viewing on all devices.
- **Navigation**: Easy-to-use navigation menu with mobile support.
- **State Pages**: Dedicated pages for each participating state.
- **SEO Optimized**: Basic SEO setup with proper meta tags and structured data.

## Structure

- `index.html`: Homepage.
- `about.html`: About Us page.
- `mandate.html`: Project Mandate page.
- `advertisements.html`: Advertisements page.
- `news.html`: News page.
- `participating-states.html`: List of participating states.
- `gallery.html`: Gallery page.
- `project.html`: Projects page.
- `resources.html`: Resources page.
- `contact.html`: Contact page.
- `states/`: Directory containing individual state pages (e.g., `states/adamawa.html`).
- `css/`: Custom CSS styles.
- `js/`: JavaScript files.
- `images/`: All images used on the site.

## Getting Started

To run this website locally:

1.  Clone the repository.
2.  Open `index.html` in your web browser.

For deployment, ensure the server supports HTML5 routing or configure the server to handle requests for pages without extensions (e.g., using `.htaccess` for Apache or Nginx configuration).
